# 성장 전략 — Atomic Network 우선

**근거**: [knowledge-wiki/_Methodology/viral_growth_principles.md](https://github.com/Lonzino/knowledge-wiki) (1차·학술 검증된 원리 적용)

**핵심 결론**: 사이트를 더 만들기 전에 **하나의 작은 씬**을 100% 점유부터.

## 왜 이 방향인가

### 검증된 원리 (1차 출처)
1. **Andrew Chen (a16z, *Cold Start Problem*)**: 전체 시장 X → atomic network 한 곳씩 → 검증 후 복제
2. **Andrew Chen**: "Retention causes virality" — 신규 유입보다 머무는 사용자가 우선
3. **Stewart Butterfield (Slack)**: "한 명만 거부하면 그 팀 끝" — 그룹 도구는 첫 1명 UX가 절대적
4. **HBR 2019 (MIT Sloan)**: clustering이 강한 시장에선 분화된 atomic network 전략 효과적
5. **KISDI 2024**: 한국 SNS 1위 인스타 45.8% (Z·M세대 절반 이상 쏠림)

### 우리 상황
- 지금 사용자 0
- 데이터 11배틀 / 10크루 (다양한 지역 분산)
- 코드는 충분 (참가 신청·체크인까지)
- **결핍**: 한 atomic network에서 검증된 사용

## 전략

### Phase B-1 (현재 → 2주): Atomic Network 1개 정복

**선택 후보 3가지** (본인 결정 필요):

| 후보 | 이유 | 위험 |
|------|------|------|
| **A. 광주 LINE UP 씬** | 빛고을댄서스가 시즌 11까지 운영 (검증된 정기대회), 멀티 장르(힙합/팝핑/락킹/왁킹) | 운영자(본인) 광주 접근성 |
| **B. 서울 비보잉 씬** | R16·BBIC 등 메이저 + 진조크루 등 알려진 크루 다수 | 큰 씬이라 atomic이 아님 (전체 점유 어려움) |
| **C. 부천 힙합 페스티벌 1회성** | 2026-05-04 임박, 단일 이벤트로 검증 | 정기성 부족, 한 번 끝나면 재방문 동력 약함 |

**추천**: **A (광주 LINE UP)**.
- 정기성 (시즌 11까지 = 충성 사용자 풀 존재)
- 멀티 장르로 다양성 검증
- 빛고을댄서스 한 곳만 설득해도 atomic network 형성 가능

### 검증 기준 (Phase B-1 성공의 정의)
| 지표 | 목표 |
|------|------|
| 광주 씬 댄서 가입 | **30명+** |
| 그 씬의 다음 배틀 사이트 등록 | **100%** |
| 그 배틀의 신청 사이트 통과 비율 | **50%+** |
| 결과 발표 후 재방문 | **70%+** |

## 코드보다 사람 먼저 (지금 진짜 해야 할 것)

### 1. 빛고을댄서스 운영자 컨택 (이번 주)
- 인스타 DM 또는 직접
- 메시지 초안:
  > "안녕하세요. 한국 스트릿 댄스 배틀 정보 사이트를 만들고 있는 Lonzino입니다.
  > LINE UP 시즌 11이 6월에 있는데, **신청·정산·체크인까지 사이트로 받으시면 운영 부담을 줄일 수 있는 무료 도구**가 있어요.
  > 베타 사용자로 시즌 11 한 번 같이 운영해보시면, 부족한 부분 직접 듣고 보강해드립니다.
  > 30분 통화나 광주 한 번 가능할까요?"

- 거부되면: 다음 후보 (CIDC, 청소년 페스티벌 등)
- 첫 1명 (이 경우 빛고을댄서스 운영자) 만족이 모든 것

### 2. 광주 댄서 직접 만남 (2주 안)
- LINE UP 시즌 11 신청 페이지 같이 만들기 워크숍
- 카톡 단톡방 또는 디스코드 운영
- 본인 직접 운영 — 자동화 X (지금 단계)

### 3. 그들의 피드백을 매주 코드 수정
- 한 줄도 가설로 만들지 말고 **그들이 원하는 것**만
- "있으면 좋은 거" X, "없으면 안 쓸 거"만
- 인터뷰 1회 = 1주일 코드 작업의 가치

- [ ] 광주 LINE UP 시즌 11 페이지 풀 입력 (부문·일정·장소·결제 안내)

## 잘 안 되면 (포기 신호)

다음 신호가 동시에 4주 이상이면 atomic network 재선택:
- 광주 댄서 가입 < 10명
- LINE UP 운영자가 사이트 사용 거부
- 신청보다 외부 폼이 더 편하다는 피드백 다수
- 본인이 광주 직접 가는 게 비합리적

이 경우: 후보 B (서울 비보잉) 또는 C (단일 이벤트 검증)으로 전환.
